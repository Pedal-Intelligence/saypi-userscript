Say, Pi

Enhance your voice interactions with Inflection AI's "Hey, Pi" chatbot using the "Say, Pi" application.

Description:

The "Say, Pi" application allows you to have immersive spoken dialogues with Pi, the intelligent AI chatbot from Inflection AI's "Hey, Pi" platform. It enhances the voice dictation capabilities, enabling accurate voice transcription and real-time interactions with Pi.

Demo:

Watch the demo video below to see how the "Say, Pi" application works.

How to Install:

  1. Install a userscript manager extension for your browser: Userscript managers are browser extensions that allow you to run custom userscripts on web pages. Some popular userscript managers include Tampermonkey and Greasemonkey. Choose the one that's compatible with your browser and install it from the extension/add-on store.
  2. Install and Enable the "Say, Pi" Userscript: Click on saypi.js. Your userscript manager will detect the file as a userscript and prompt you to install it. Confirm the installation to enable the "Say, Pi" userscript for the "Hey, Pi" web platform.
  3. Visit the "Hey, Pi" web platform: Go to https://heypi.com/ using your browser. The "Hey, Pi" platform is where you'll use the "Say, Pi" userscript to interact with Pi, the intelligent AI chatbot.
  4. Permissions: You will be prompted to grant the script permission to access your microphone, and to make requests to this website (to load resources and transcribe audio). Review and accept the permissions to continue.
  5. Automatic Injection: Once you have the userscript manager extension installed and the "Say, Pi" userscript enabled, it will automatically inject a "talk" button onto the "Hey, Pi" web interface. You'll see the "talk" button appear on the page.
  6. Start Speaking with Pi: To start a spoken dialogue with Pi, simply click the "talk" button and hold it while speaking. The userscript will transcribe your speech in real-time and send it to Pi, who will respond with both text and audio.

Compatibility:

The "Say, Pi" application is compatible with popular browsers such as Chrome and Edge. Please ensure you have a userscript manager extension installed in your browser before using it.
