Say, Pi Application

Enhance your voice interactions with Inflection AI's "Hey, Pi" chatbot using the "Say, Pi" application.

Description:

The "Say, Pi" application allows you to have immersive spoken dialogues with Pi, the intelligent AI chatbot from Inflection AI's "Hey, Pi" platform. It enhances the voice dictation capabilities, enabling accurate voice transcription and real-time interactions with Pi.

How to Install:

  1. Install a userscript manager extension for your browser (e.g., Tampermonkey, Greasemonkey).
  2. Visit the "Hey, Pi" web platform at https://heypi.com/.
  3. Once on the "Hey, Pi" website, the "Say, Pi" userscript will automatically inject a "talk" button onto the web interface.
  4. Click the "talk" button to initiate a spoken dialogue with Pi. Hold the button while speaking, and Pi will respond with both text and audio.

Compatibility:

The "Say, Pi" application is compatible with popular browsers such as Chrome and Edge. Please ensure you have a userscript manager extension installed in your browser before using it.

Disclaimer:

The "Say, Pi" application is an unofficial enhancement for the "Hey, Pi" web platform. Use it responsibly and respect the terms of service of "Hey, Pi" provided by Inflection AI. We cannot guarantee its compatibility with future updates or changes to the "Hey, Pi" platform.
